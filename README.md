📊 PlanilhaCerta

PlanilhaCerta — livros de Excel + planilha grátis para acelerar sua produtividade.

🚀 Sobre o projeto

Este repositório contém o código do site PlanilhaCerta, uma landing page criada para apresentar e vender materiais digitais focados em Excel, além de oferecer uma planilha gratuita como isca para atrair e captar contatos de interessados.

📚 Produtos disponíveis
Excel para Iniciantes: Aprenda a Criar Planilhas, Usar Fórmulas e Organizar sua Rotina
Controle de Estoque no Excel: Planilha Completa com Alertas Automáticos
Dashboards Profissionais no Excel (nível iniciante → intermediário)
Excel para Controle de Vendas (comissões + metas)
Excel para Controle Financeiro Pessoal
Excel para MEI – Controle de Vendas, Caixa e Lucro de Forma Simples

Todos os livros estão publicados na Amazon.

🎯 Objetivo do site
Apresentar os livros de forma clara e atrativa
Direcionar usuários para compra na Amazon
Capturar leads através de uma planilha gratuita
Demonstrar na prática o valor do Excel no dia a dia
🛠️ Tecnologias utilizadas
HTML
CSS
JavaScript
📬 Captação de leads

O site oferece uma planilha gratuita em troca do e-mail do visitante, com o objetivo de:

construir uma base de contatos
nutrir leads com conteúdo futuro
aumentar as vendas dos produtos
🌐 Acesso

O site pode ser acessado via GitHub Pages.

📌 Observações

Este projeto faz parte de uma estratégia de marketing digital voltada para educação em Excel e venda de produtos digitais.
