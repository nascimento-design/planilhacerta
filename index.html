Dúvidas

Perguntas frequentes

Preciso ter experiência em Excel para usar os e-books?
Não! Os e-books de nível iniciante foram escritos pensando em quem nunca abriu o Excel antes. Você aprenderá do zero, com explicações simples e exemplos práticos.
As planilhas gratuitas já vêm prontas para usar?
Sim! As planilhas são entregues já configuradas, com fórmulas e formatação prontas. Você só precisa inserir seus dados e elas já calculam tudo automaticamente.
Em qual formato os e-books são entregues?
Os e-books disponíveis na Amazon são entregues no formato Kindle (para leitura pelo app gratuito da Amazon, disponível para celular, tablet ou PC). Os demais serão entregues em PDF.
Posso usar as planilhas no Excel do celular?
Sim! As planilhas são compatíveis com o Excel do computador e com o Microsoft Excel para Android e iOS. Para uso básico, o app gratuito do celular já é suficiente.
Os e-books "Em breve" têm data de lançamento?
Estamos finalizando a publicação. Quem baixar as planilhas gratuitas será notificado por e-mail assim que cada título estiver disponível — geralmente com desconto de lançamento.
